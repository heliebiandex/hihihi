小九的发展历程
小九是我在一念之间制作出来的，它可以进行简单的对话操作，不需要连接外网，是用VBScript Script(VBS)做出来的。
刚开始时，他还有一个小程序，是贪吃蛇，代码从网上下载。
我用了大概2个月，从暑假一直忙到现在，估计后面还会更新。
关于我刚才提到的一念之间：
我给你们讲一个故事吧。有一次，在我家里，突然，一楼跳闸了。因为整座楼的供网（internet）
都是在一楼，所以几乎同一时间，网也断了。当时我在看bilibili，就突然弹出个窗口说没连接
到internet。当时我就慌了：好端端的应该不会断网。一看才知道，一楼跳闸了。
我当时就很无聊，就只能打开几个自己做的vbs来玩。到后来，我就想起来了：
既然人人家里都会停电、没网，那我是不是就可以利用这个问题做一个小程序呢？
说干就干。很快，在查阅了无数次的w3school后，
小九1.0做好了。当时就是那个我说过的有小程序的版本。
然后，我就想了:打开小程序需要用绝对路径，那么就会暴露我自己。
所以，在删了几行代码后，小九1.2———也就是现在这个版本，诞生了。
